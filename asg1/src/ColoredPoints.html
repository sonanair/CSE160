The "awesomeness" I added were an Undo and Redo button. This way, if you do not want to clear your entire screen and just erase the most recent shape, the undo button is useful. Likewise, the redo button reverses the undo action. To accomplish this, I created two new functions, one for each respective action. The Undo function pops the most recent action stored in the g_shapesList, stores the shape in a new stack, and re-renders the rest of the shapes. The Redo function takes the stored shape and adds it back to g_shapesList, restoring the original drawing.
